HTML이란?

HTML은 웹에서 사용하는 문서를 만드는 언어입니다.

CSS란?

문서를 시각화하기 위해 웹 상에서 약속한 언어가 바로 CSS입니다.

자바스크립트란?

웹브라우저에서 실행하는 스크립트 언어입니다. 또한, 웹 브라우저에서 동적인 페이지를 만들기 위해 탄행하였습니다.

제이쿼리란?

자바스크립트의 생산성을 향상시켜 주는 자바스크립트 라이브러리로, 자바스크립트를 좀 더 쉽게 사용하기 위해 만들어 졌습니다.

Ajax란?

Asynchronous JavaScript and XML의 약자로 자바스크립트를 이용해서 비동기식으로 서버와 통신하는 방식을 말합니다.

느낀점 IT 영재캠프 참여 후 느낀점을 이야기해 보았습니다.

  1. Q1디자인 수업 후 느낀점

    다연 : 디자인 수업은 원래 꾸미는 것도 잘 못하는지라 누끼 따는 것까진 어떻게 하면 할 수 있긴 했는데, 배치하고 이런 것들이 너무 어려웠어요... 어떻게 하면 잘 배치할 수 있을까 고민했는데 완성물이 좀 지저분해서 마음에 들진 않지만 ㅠㅠ 그래도 잘 가르쳐주셔서 감사했어요!

    수하 : 누끼따는 것이나 디자인 한 것이 생각보다 어려워서 고민을 많이 했던 것 같습니다. 하지만 디자인 수업을 듣고 제가 직접하면서 정말 즐거웠고 결과물을 보면서 뿌듯함도 느낄 수 있었던 것 같습니다.

    은채: 좋아하는 디자인을 색다른 방법으로 할 수 있어서 즐거웠어요. 만족스러운 결과물이 나와서 뿌듯하네요!

    은채 : 손으로 하는 것보다는 쉬울 줄 알았는데 어려웠어요. 사진고르는 것 부터 생각보다 복잡했어요. 앞으로도 연습해보려고요!

  2. Q2SW수업 후 느낀점

    다연 : 개발 수업은 입력해야 할 것도 많았고, 조금 횡설수설했지만 하나하나 해볼 때마다, 알아갈 때마다 너무 재밌어져서 좋았어요! 완성물 보니까 뿌듯하기도 하고...

    수하 : 코딩이 어렵고 복잡한 것을 알고 있었지만 직접 해보니 생각했던 것 보다 더 복잡하고 하는 내내 많은 어려움이 있었지만 멘토 언니 덕분에 잘 이겨낼 수 있었고 결과물도 맘에 쏙 들게 나온 것 같아서 너무 좋았습니다

    은채 : 개발은 처음이라 걱정했지만 멘토 언니가 차근차근 알려주셔서 수월하게 해낼 수 있었어요! 생각보다 즐거웠고 새로운 흥미를 찾을 수 있었네요.

    은채 : 시작부터 영어가 나오길래 어려울 줄 알았는데 생각보다 쉬웠어요. 단어가 조금만 달라져도 결과가 휙휙 달라져서 신기했어요.

  3. Q32020 it영재캠프에 참여 후 느낀점

    다연 : 원래 그냥 컴퓨터 다루는 걸 좋아하기도 했고, 단축어 외우는 거나 입력하는 걸 재밌어해서 무작정 '미림은 유일한 IT 여고입니다!'라는 걸 보고 캠프 지원까지 했는데, 디자인은 좀 어려웠지만 프로그래밍 개발은 재밌었어요! 정말 꼭 들어가고 싶어요 ㅠㅠ

    수하 : 이번에 코로나 때문에 온라인으로 바뀌면서 많은 걱정도 있었고, 내가 과연 이번 캠프를 잘 마무리 할 수 있을까 고민도 많았지만 왜 그런 고민을 했을까 싶을 정도로 생각했던 것보다 더 재미있었고 좋은 시간들이였던 것 같습니다. 선생님과 멘토 언니들 덕분에 많은 것들도 배우고 갈 수 있어서 좋은 것 같습니다.

    은채 : 디자인만 보고 신청한 캠프인데 프로그래밍을 더 열심히 했어요. 새 적성을 찾은 것 같기도 하고, 쉽사리 오지 않는 기회라 정말 의미있는 시간이었습니다. 여러모로 유익했어요! 신청하길 정말 잘한 것 같고 한번뿐이라 아쉽네요. 저희를 위해 힘써주신 선생님들과 멘토 분들 정말 감사합니다!

    은채 : 평소에 잘 접하지 못한 활동을 해봐서 좋았어요. 멘토 언니들이랑 선생님들이 잘 알려주셔서 감사했어요.
